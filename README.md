# Análise Estatística do Álbum da Copa do Mundo 2026

## Objetivo

Este projeto tem como objetivo analisar, através de simulações e técnicas de análise de dados, o custo estimado para completar o álbum oficial da Copa do Mundo 2026.

A proposta é utilizar Python para modelar a compra de pacotes de figurinhas, calcular estatísticas e visualizar padrões relacionados à coleção do álbum.

---

## Informações do Álbum

* Total de cromos: 980
* Cromos normais: 912
* Cromos brilhantes: 68
* Figurinhas por pacote: 7
* Preço por pacote: R$ 7,00
* Preço estimado do álbum capa dura: R$ 75,00

---

## Objetivos da Análise

O projeto busca responder perguntas como:

* Quantos pacotes são necessários, em média, para completar o álbum?
* Qual o custo médio total da coleção?
* Quantas figurinhas repetidas surgem ao longo do processo?
* Em qual momento as repetidas começam a dominar os pacotes?
* Quanto as trocas entre colecionadores podem reduzir os custos?

---

## Metodologia

Inicialmente, a simulação considera que todas as figurinhas possuem a mesma probabilidade de aparecer nos pacotes.

Posteriormente, o projeto poderá incluir:

* probabilidades diferentes para figurinhas brilhantes;
* modelagem de raridade;
* análise das figurinhas Legends/Extras;
* simulações de troca entre colecionadores.
